Welcome to the Maze Project

The goal of this project is to create a 3D maze game using raycasting. You'll develop a game where the camera navigates through walls, starting with SDL2 basics and advancing to textures, enemies, and effects. Follow the tasks, dive into the tutorials, and enjoy the process!
